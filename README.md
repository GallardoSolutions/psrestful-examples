# psrestul-examples

Showcasing examples on how to use [PSRESTful API](https://psrestful.com), an API based on PromoStandards.

- [Order Status Example](order_status/README.md)
- [Real-time Inventory Example](inventory/README.md)


## Documentation

- [PSRESTful Documentation](https://docs.psrestful.com/)
- [PSRESTful API Reference](https://api.psrestful.com/docs/)
- [PSRESTful blog](https://blog.psrestful.com/)


## Technologies used

### In order to be simple enough to be used as an example, we used the following technologies:

- [FastAPI](https://fastapi.tiangolo.com/)
- [JQuery](https://jquery.com/)
- [Bootstrap](https://getbootstrap.com/)
- [Jinja2](https://jinja.palletsprojects.com/en/2.11.x/)
- [Htmx](https://www.htmx.org/)
- [Alpine.js](https://alpinejs.dev/)
